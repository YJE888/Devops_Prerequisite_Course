### hosting
- 웹을 실행하고, 사용자에게 서비스를 제공하는 것
- 웹 서버에 있는 `Application Code`를 제공하는데 사용되는 소프트웨어는</br> `Apache HTTP Server`(정적인 서버에 사용)와 `Apache Tomcat`(동적인 서버에 사용) 등이 있음
- 하나의 서버에서 여러 개의 웹이 구동 가능하며, 이는 ***URL*** 또는 ***Hostname***으로 구분함
